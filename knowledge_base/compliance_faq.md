# Security and Compliance FAQ

## HIPAA Compliance
* The platform fully supports healthcare operations and signs Business Associate Agreements (BAAs) with Enterprise tier clients.
* All data is robustly encrypted at rest using AES-256 and in transit using TLS 1.3.

## GDPR and Privacy Frameworks
* Under GDPR Article 20, users possess a formal Right to Portability. Upon explicit request, a complete data export must be delivered to the client in a structured, machine-readable JSON format within 30 statutory days.
* Data Processing Agreements (DPAs) are available for digital download via the billing profile portal.

## Certifications and Datacenters
* The infrastructure maintains active SOC 2 Type II and ISO 27001 certifications.
* Multi-region hosting options are restricted to AWS data centers based in US-East, EU-West, and Singapore.
